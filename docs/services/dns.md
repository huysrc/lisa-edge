# DNS Helpers

DNS helpers provide stable local names for internal services.

## Purpose

Examples:

- `mqtt.home.arpa`
- `lisa-edge.home.arpa`
- `uptime.home.arpa`
- `lisa-brain.home.arpa`

## Recommendation

Prefer your gateway or existing DNS server as the source of truth.

Use LISA Edge only for lightweight DNS helper roles unless you intentionally choose it as your DNS service node.
