This directory contains Ansible playbooks for automating the deployment and configuration of LISA Edge services. The playbooks are designed to be run on a Linux system and require Ansible to be installed.

The playbooks are organized into separate files for each service, and each file contains the necessary configuration and deployment steps for that service. The playbooks are written in YAML and use Ansible's built-in modules to automate the installation and configuration of the services.

---
### NOTE
playbooks/ is experimental and not aligned with the current /opt/lisa-edge + /srv/lisa-edge layout yet.
